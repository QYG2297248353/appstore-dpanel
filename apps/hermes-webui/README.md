## 简介

Hermes Agent是一个复杂的自主代理，它位于您的服务器上，通过终端或消息应用程序访问，它会记住它所学到的内容，并且运行的时间越长，它的能力就越强。

爱马仕WebUI是爱马仕代理浏览器中的一个轻量级、黑暗主题的web应用界面。与CLI体验完全一致——您可以在终端上做的一切，都可以在这个UI上做。没有构建步骤，没有框架，没有捆绑器。只有Python和普通JS。
