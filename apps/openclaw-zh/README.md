## 简介

> “去壳！去壳！” — 大概是一只太空龙虾说的

OpenClaw 通过单个 Gateway 网关进程将聊天应用连接到 Pi 等编程智能体。它为 OpenClaw 助手提供支持，并支持本地或远程部署。
