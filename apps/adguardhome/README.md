## 简介

AdGuard Home 是一种基于网络的广告和跟踪器拦截解决方案。只需在您的路由器上安装一次，即可覆盖您家庭网络中的所有设备——无需额外的客户端软件。这对于各种经常威胁您隐私的物联网设备尤其重要。

## 安装说明

需要提前关闭宿主机 `resolved` 功能，否则会导致 DNS 解析失败。

```sh
systemctl disable systemd-resolved
systemctl stop systemd-resolved
```
