## 简介

一个免费开源的漫画阅读器服务器，运行为 Mihon（Tachiyomi） 构建的扩展程序。

Suwayomi 是一个独立的 Mihon (Tachiyomi) 兼容软件，而不是 Mihon (Tachiyomi) 的 Fork 。

Suwayomi-Server 是一款真正意义上的多平台应用。
