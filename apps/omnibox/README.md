## 特性

+ 支持采集站、网盘资源在线观看
+ 支持 IPTV 源观看
+ 支持 6 大直播间平台观看
+ 支持解析站解析爱优腾主流视频站链接
+ 支持网页、移动端、TV 端观看。
+ 各个端都支持弹幕功能
+ 在线看 4K 视频
